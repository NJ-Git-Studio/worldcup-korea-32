# 🇰🇷 월드컵 2026 · 한국 32강 진출 경우의 수

FIFA 공식 데이터를 실시간으로 가져와 **대한민국의 32강(라운드 32) 진출 경우의 수**를
계산·시각화하는 로컬 웹 대시보드입니다.

2026 월드컵은 48개국 체제라 **각 조 1·2위(24팀) + 3위 중 상위 8팀 = 32팀**이 진출합니다.
따라서 한국이 조 3위로 마치면 *다른 11개 조의 3위 팀들과의 와일드카드 경쟁* 결과에 따라
진출이 갈리며, 이 프로그램이 그 경우의 수를 자동으로 따져 줍니다.

---

## 빠른 시작

```bash
pip install -r requirements.txt
python app.py
```

브라우저에서 **http://127.0.0.1:5000** 접속.

> Python 3.9+ 권장. 인터넷이 되면 FIFA 공식 API에서 최신 결과를 가져오고,
> 실패 시 openfootball 공개 데이터 → 로컬 캐시 순으로 폴백합니다.

## 주요 기능

| 기능 | 설명 |
|------|------|
| **진출 판정** | 진출확정 / 경합 / 탈락을 색상 배지로 표시 |

| **미완 조별 위협 분석** | 각 미완 조의 3위가 한국을 밀어낼 확률을 막대로 시각화 |

| **🔄 갱신** | 버튼 한 번으로 FIFA 최신 결과 다시 불러오기 |

## 데이터 소스

1. **FIFA 공식 API** — `api.fifa.com/api/v3/calendar/matches`
   (competition `17` = FIFA World Cup, season `285023` = 2026)
2. **openfootball** — `github.com/openfootball/worldcup.json` (폴백)
3. **로컬 캐시** — `cache/` 에 저장된 마지막 스냅샷 (오프라인 폴백)

우선순위는 환경변수 `WC_PREFER` 로 바꿀 수 있습니다 (`fifa` | `openfootball` | `cache`).

### 수동 입력 / 보정

네트워크가 안 되거나 특정 경기 결과를 직접 넣고 싶으면 프로젝트 루트에
`manual_matches.json` 을 만들면 같은 경기의 점수를 덮어씁니다.

```json
[
  { "group": "A", "home": "Mexico", "away": "Korea Republic",
    "home_score": 1, "away_score": 0 }
]
```

## 다른 사람과 공유하기 (공개 링크)

`cloudflared` 터널로 내 PC의 대시보드를 **임시 공개 URL**로 노출할 수 있습니다.
회원가입이 필요 없습니다.

```bash
# 최초 1회: cloudflared 설치 (Windows)
winget install Cloudflare.cloudflared
```

설치 후에는 프로젝트 폴더의 **`share.bat` 더블클릭** 한 번이면 됩니다.
- 대시보드 서버 + 터널이 함께 켜지고, 창에 `https://....trycloudflare.com` 주소가 표시됩니다.
- 그 주소를 카톡/메일로 보내면 누구나 접속해 빙고판·What-if를 사용할 수 있습니다.

수동으로 하려면 두 개의 터미널에서:

```bash
python app.py
cloudflared tunnel --url http://localhost:5000
```

> 주의
> - 내 PC가 켜져 있고 `share.bat`/터널이 실행 중인 동안에만 접속됩니다.
> - 실행할 때마다 주소가 새로 발급됩니다(고정 아님).
> - 링크를 받은 사람은 누구나 접속할 수 있으니 공개 범위에 유의하세요.
> - 24시간 고정 주소가 필요하면 Render/Railway 같은 무료 클라우드 배포를 사용하세요.

## 진출 규정 / 타이브레이커

- **승점**: 승 3 · 무 1 · 패 0
- **조 내 순위**: 승점 → 골득실 → 다득점 → (동률 시) 승점·골득실·다득점(맞대결) → 페어플레이 → 추첨
- **3위 팀 간 순위**: 승점 → 골득실 → 다득점
  - 페어플레이/추첨 단계는 데이터가 없어 모델링하지 않으며,
    승점·골득실·다득점이 모두 같은 경우는 "경합"으로 표시합니다.

- `POST /api/whatif` — `{ "outcomes": { "<match_id>": "home|draw|away" } }`

## 동작 원리(경우의 수 판정)

한국 조가 끝나 3위로 확정되면, 다른 각 조는 서로 **독립**이므로 조별 잔여경기를
전수조사해 "그 조의 3위가 한국보다 위가 *될 수 있는가 / 반드시 위인가*"를 판정합니다.

- 한국보다 위가 *반드시* 되는 3위가 **8팀 이상** → **탈락 확정**
- 한국보다 위가 *될 수 있는* 3위가 **7팀 이하** → **진출 확정**
- 그 사이 → **경합** (몬테카를로 확률 + What-if 로 구체화)

모든 경기가 끝난 경우에는 실제 3위 랭킹으로 정확히 진출/탈락을 판정합니다.
